HelloJS 八週心得

學習技術統整

  • GIT版控
  • npm
  • hexo
  • api
  • sails
  • node.js
  • React + Redux + React Native
  • Jenkins and Docker

心得總結

這八週課程內容對我來說是全新的，因為目前在做畢業專題而來到這課程學習。
每個禮拜一次的課程時間 剛剛好，因為沒有認識的同學陪伴，就決定先當個邊緣人好了
同校的人滿多的，雖然跟自己不同年級，但怎麼想都有一種親切感，加上講師跟助教都是差不多年紀的人
緩解了許多緊張~~ 整體上課氣氛是輕鬆和樂的。

關於上課內容

對第一接觸的git版控很有心得，是團隊開發很重要的一個環節，
雖然對於分支跟衝突的概念還是沒有很完全接收，但在課程上不斷使用基本的 add.commit.push.clone.
在最後demo實作的過程中 如何更新repo資料

經過了git 在接下來老實說進入一種未知的狀態…..
全新探索未知的世界，基礎不太OK的我….真的處於迷茫
唯一比較有趣的是hexo 算是玩得比較快樂的地方 起碼會架設會基本使用
後來一連串的新技術 就通通接收 但對於使用應該是存在著疑問的
但還是會努力 去使用到會更熟悉的~~~
原本想要每個禮拜更新一點東西 馬上hexo提醒自己
但都亂糟糟的躺在桌面上 接下來是整理好時間 順便總結一下八堂課

關於demo成果實作

這次demo一個吉他轉調系統，專門設計給吉他新手跟團體教學，因為本身在社團裡只會基本和弦
但想彈的歌曲有很多，很多和弦都要重新學習跟練習，為了增加練習時的信心，因而嘗試開發APP
雖然自始至終對於後端跟前端的連結都不太熟悉，但跟著隊友學習還是得到很多製作的技巧跟技術
因為版面需要一點美編的圖片，很開心在這一方面可以貢獻一點己力。
看著成果code 自己當然還有很大的空間要去摸索….
在陰錯陽差下找到的隊友 ((原本想在這次實作demo上消失XD
還有被接收 被凱瑞 就算厚臉皮還是硬著頭皮跟著學習
非常感謝隊友

關於創科

好給立 好專業 好環境 好教學 好飲料 好wifi 好冷氣
來了第二次 雖然空間真的滿小XDD 但好像不會真的覺得不好
期待之後繼續有棒棒的課程
我會持續關注
感謝裏頭各個都高個兒的大家 ((都很高…這實在終究成為我的疑問))
